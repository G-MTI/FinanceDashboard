# Expense Tracker

A simple web app to track income and expenses and to visualize them with charts.
For reviewers you need to add category befor try to add a transaction

Users can:
- Add transactions (income / expense)
- Delete transactions
- Modify categories with custom colors

Tech Stack
 - React (Vite)
 - Tailwind CSS
 - Recharts

The data are saved in localStorage in future Improvements I'll make it possible to save it with the account.
